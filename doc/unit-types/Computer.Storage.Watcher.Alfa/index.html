Computer Storage Unit

Description

The unit periodically gets metrics of the operating system's storage and places them in the appropriate data items.

Parameters

Data Items

The unit writes the measured parameters to the following data items:
  • SpaceTotal
  • SpaceFree
  • SpaceUsed
  • BlocksTotal
  • BlocksFree
  • BlocksUsed
  • INodesTotal
  • INodesFree
  • INodesUsed
  • SpaceUsedPercents
  • BlocksUsedPercents
  • INodesUsedPercents
  • Status
