Ping unit

Description

The unit sends ICMP-requests to the remote host and measures the response time. The measured time is written to the data item. It supports IPv4 only. This unit allows you to monitor the network availability of devices.

Parameters

Address
Destination host. It can be a domain name or a IP-address.
FrameSize
Size of a data block. Minimal size is 1 byte. Maximal size is 1400 bytes. The data block content is "ABCDE....XYZABCDEF.....". Optimal value of the parameter is 64 bytes.
Timeout
Timeout in milliseconds to wait for each reply. Minimal size is 100 ms. Maximal size is 10000 ms (10 seconds). Optimal value of the parameter is 1000 (1 second).
Period
The period between sensor activities. Minimal size is 100 ms. Maximal size is 10000 ms (10 seconds). Optimal value of the parameter is 1000 (1 second).

Data Items

Time
Response time
Address
Address from the configuration.
DataSize
Size of a data block.
