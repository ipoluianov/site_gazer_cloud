Computer Network Interface Unit

Description

The unit periodically gets metrics of the operating system's network interface and places them in the appropriate data items.

Parameters

Data Items

The unit writes the measured parameters to the following data items:
  • TotalSpeed
  • Status
  • InOctets
  • InUcastPkts
  • InNUcastPkts
  • InDiscards
  • InErrors
  • OutOctets
  • OutUcastPkts
  • OutNUcastPkts
  • OutDiscards
  • OutErrors
  • InSpeedFrames
  • InSpeed
  • OutSpeedFrames
  • OutSpeed
