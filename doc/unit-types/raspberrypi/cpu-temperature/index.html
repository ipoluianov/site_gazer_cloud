RaspberryPI CPU Temperature Unit

Description

The unit gets the temperature of the processor and writes it to the corresponding item

Parameters

Period
The period between sensor activities. Minimal size is 100 ms. Maximal size is 10000 ms (10 seconds). Optimal value of the parameter is 1000 (1 second).
