Installation on Linux / MacOS

Download

Linux x64

wget https://gazer.cloud/download/2-4-4/linux_amd64/gazer_node_amd64

Linux arm 32

wget https://gazer.cloud/download/2-4-4/linux_arm32/gazer_node_arm

Linux arm 64

wget https://gazer.cloud/download/2-4-4/linux_arm64/gazer_node_arm64

Mark as an executable file

chmod 755 gazer_node_amd64

Installation as a service

./gazer_node_amd64 -install
./gazer_node_amd64 -start

Uninstall the Gazer service

./gazer_node_amd64 -stop
./gazer_node_amd64 -uninstall
